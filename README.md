# Менеджер подписок VK Bot

Бот ВКонтакте для управления подписками с использованием библиотеки VKBottle.

## Возможности

Бот позволяет:

- добавлять подписки;
- просматривать список подписок;
- изменять стоимость подписки;
- удалять подписки;
- оплачивать подписки;
- просматривать неоплаченные подписки;
- подсчитывать общие расходы.

Для хранения данных используется JSON-файл.

## Конечный автомат

В проекте реализован конечный автомат (FSM).

Поддерживаются состояния:

* DEFAULT
* ADD_NAME
* ADD_PRICE
* DELETE_SELECT
* PAY_SELECT
* UPDATE_SELECT
* UPDATE_PRICE

FSM используется для пошагового взаимодействия пользователя с ботом.

### Добавление подписки

1. Нажмите кнопку «Добавить».
2. Введите название подписки.
3. Введите стоимость.

---

### Удаление подписки

1. Нажмите кнопку «Удалить».
2. Введите ID подписки.

---

### Изменение стоимости

1. Нажмите кнопку «Изменить».
2. Введите ID подписки.
3. Введите новую стоимость.

---

### Оплата подписки

1. Нажмите кнопку «Оплатить».
2. Введите ID подписки.

---

### Просмотр подписок

```text
Список
```
